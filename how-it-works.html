How It Works

Choose the main Stardew Valley save file, usually named like FarmName_123456789 with no extension.

The browser reads that file with the local File API, parses the XML in memory, computes recap cards, and renders the poster on your device.

Nothing in the save file is sent to a server. Closing the page clears the in-memory parsed result.

For multiplayer saves, the app lists the host and farmhands so you can pick which farmer to recap.
