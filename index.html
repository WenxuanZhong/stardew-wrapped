如何找到存档

Windows

  1. 按 Win + R 打开运行
  2. 输入 %appdata%\StardewValley\Saves 回车
  3. 找到形如 农场名_123456789 的文件夹
  4. 打开这个文件夹，选择里面无后缀、同名的主存档文件

macOS

  1. Finder 顶栏 前往 → 前往文件夹 或 Shift+Cmd+G
  2. 输入 ~/.config/StardewValley/Saves
  3. 打开农场文件夹，选择无后缀的主存档文件

Linux

  1. 路径：~/.config/StardewValley/Saves
  2. 同 macOS 操作
💡 一个农场文件夹里通常有 4 个文件：主存档（无后缀）、SaveGameInfo、以及它们各自的 _old 备份。这里只选无后缀主存档。

Stardew Valley 1.6 及以上

  1. 装一个文件管理器，推荐 MT 管理器（能进受限目录）
  2. 进入路径：
    /Android/data/com.chucklefish.stardewvalley/files/Saves/
  3. 找到农场文件夹（如 JonSnow_123456789），打开后复制同名无后缀主存档到 Download 等公共目录
  4. 用浏览器打开本页，选择这个主存档文件

较老版本 / 部分机型

  1. 路径可能直接是 /sdcard/StardewValley/Saves
  2. 普通文件管理器即可访问
💡 Android 11+ 之后 /Android/data/ 默认对普通应用关闭。MT 管理器 / Shizuku / 厂商自带文件管理可绕过。

iOS / iPadOS

  1. 打开 iPhone/iPad 自带的 "文件" app
  2. 左侧栏选 "我的 iPhone"（或 iPad）
  3. 进入 Stardew Valley 文件夹
  4. 打开 Saves 子文件夹
  5. 打开农场文件夹，找到无后缀、同名的主存档文件
  6. 在本页直接选择这个文件，或传到电脑后再选择
💡 看到 SaveGameInfo 或 _old 时先别点，主存档一般就是和农场文件夹同名、没有后缀的那个。

关于镇民头像

原版镇民头像保存在 portraits/ 文件夹（解自 Stardew Valley 的 .xnb），跟 index.html 一起部署即可显示。
